## Start
### 1.Install Cairo
```
brew install Cairo
```
### 2.Clone the repository
```
git clone https://github.com/2445868686/AiRename-Image.git
```
### 3.Install Dependencies
```
cd AiRename-Image
```
```
pip install -r requirements.txt
```
### Run
```
python main.py
```
